# Commerce System Clean Architecture

ASP.NET Core Web API built with Clean Architecture, CQRS, MediatR, FluentValidation, Entity Framework Core, Repository Pattern

## Technologies
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server
- MediatR
- FluentValidation
- Swagger
- xUnit

## Architecture
- Api
- Application
- Domain
- Infrastructure
- Common
- Tests

## Features
- Product Category CRUD
- Validation Pipeline
- Repository Pattern
- Unit of Work
- EF Core Migrations

## How to Run
1. Update connection string in `appsettings.json`
2. Run migrations
3. Start API
4. Open `/swagger`
